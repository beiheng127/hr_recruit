# HR招聘管理系统 — 项目记忆

## 项目概览

HR招聘管理系统，Spring Boot + Vue 3 全栈项目，含 AI 功能（DeepSeek API）。
实训课程项目，当前已完成核心功能开发和大部分 bug 修复。

## 技术栈

- **后端**: Spring Boot 3.2 + JDK 17 + Maven + MyBatis-Plus + Spring Security(JWT) + Flyway + Spring AI
- **前端**: Vue 3 + Vite + Pinia + Element Plus + ECharts + Axios
- **数据库**: MySQL 8.0 (19张表)
- **AI**: DeepSeek (通过 Spring AI ChatClient)

## 关键架构决策

1. **API路径**: 所有 Controller 使用 `/api/` 前缀，前端 Axios baseURL 也是 `/api`
2. **统一返回格式**: `Result<T>` 包含 code/data/message
3. **分页**: MyBatis-Plus `Page<T>`，前端用 `pageNum/pageSize` 参数，数据从 `records` 字段取
4. **认证**: JWT Token，通过 axios 拦截器注入 Authorization header
5. **角色权限**: ADMIN/HR/MANAGER 三级，前端路由守卫 + 后端 @PreAuthorize
6. **状态枚举**: 全部使用英文大写值（如 PUBLISHED, COMPLETED），前端映射为中文标签
