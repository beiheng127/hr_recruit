## 🚀 快速启动步骤

### 1. 环境准备
```
- JDK 17+
- Node.js 18+
- MySQL 8.0+
- Maven 3.6+
```

### 2. 数据库
```sql
-- 创建数据库
CREATE DATABASE hr_recruit CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
-- Flyway 会在应用启动时自动执行迁移脚本
```

### 3. 后端启动
```bash
cd hr_recruit/hr-recruit-backend
cp application.yml.example application.yml  # 编辑填入MySQL密码和AI API Key
mvn spring-boot:run
```

### 4. 前端启动
```bash
cd hr_recruit/hr-recruit-frontend
npm install
npm run dev
```

### 5. 登录测试
```

```

## ⚠️ 常见陷阱速查

| # | 陷阱 | 说明 |
|---|------|------|
| 1 | `positionName` ≠ `name` | JobPosition 岗位名字段是 `positionName` |
| 2 | 面试关联链 | Interview→stageRecord→applicationJob→(resume+job) |
| 3 | LambdaQueryWrapper空条件 | update/delete时空wrapper=操作全表！ |
| 4 | Invalid end tag行号 | Vue报错行号≠实际有多余标签的位置 |
| 5 | Map.of()不允许null | 改用HashMap并做null检查 |
| 6 | PDFBox 3.x API不同 | 用`Loader.loadPDF()`非`PDDocument.load()` |
